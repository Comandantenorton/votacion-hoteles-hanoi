# Votación hoteles Hanoi - Viaje Vietnam 2026

Página estática para votar hoteles en Hanoi del 27 al 30 de septiembre de 2026.

## Uso

Publicar este repositorio con GitHub Pages desde la rama `main` y la carpeta `/root`.

La página no usa backend ni guarda respuestas automáticamente. Al enviar, genera un mensaje de WhatsApp con el voto completo.
